Clusters -- Далее идет инфа о кластерах

-- Кластеров можно лепить изначально сколько захотите
-- Random - значит координаты будут случайными (нужно задать пределы целыми числами)
-- форматы следующие
-- 1) Random <Bound1> <Bound2> [Center Id] [Cluster Id], 
--	где Bound1, Bound2 - диапазон целых знайчений для координат центра кластера
--	и, да, для точек такой не канает
-- 2) <X> <Y> [Center Id] [Cluster Id], 
--	где X, Y - координаты цетра кластера

Random -1 12 Z1 -- случайный центр с заданой айдишкой
0 6 Z2 S2 -- полностью заданый кластер

Config -- Параметры кластеризации (вводить внимательно названия)

Nmin 1 -- минимальное количество точек, которые могут формировать кластер
Imax 5 -- максимальное кол-во итераций (5ти хватает с головой, даже много)
Dmax 1.9 -- максимальное отклонение точки от центра кластера
Lmin 2.5 -- минимальное расстояние между 2мя кластерами (их центрами собственно)
Pmax 10 -- кол-во пар(!) кластеров которые можно объединить за одну итерацию

-- выводит лишние итерации
